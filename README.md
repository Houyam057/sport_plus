# Sport Plus - PFA Project

## Tech Stack
- PHP
- MySQL
- HTML/CSS

## Setup Instructions

1. Clone the repository
2. Move project to htdocs (XAMPP)
3. Start Apache and MySQL in XAMPP
4. Open phpMyAdmin
5. Create a database (e.g. sport_plus)
6. Import database/sport_plus.sql
7. Configure database connection in your project
8. Run: http://localhost/sport_plus
